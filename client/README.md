# NextGent - Frontend Client

This is the React-based frontend for **NextGent**, built with modern design principles and high-performance technologies.

## 🛠 Tech Stack

- **React 19**: Utilizing the latest React features.
- **Vite**: Ultra-fast build tool and development server.
- **Tailwind CSS 4.0**: Powering the advanced dark-mode UI system.
- **Framer Motion**: Smooth page transitions and interactive card animations.
- **Lucide React**: Clean, consistent iconography.
- **React Router**: Client-side routing for seamless navigation.

## 📁 Key Directories

- `src/components`: Core feature components (UserProfile, UserOutput, UserDashboard).
- `src/UI`: Reusable UI primitives (Buttons, Viewers, Modals).
- `src/pages`: Top-level page definitions (LandingPage, Auth).
- `src/assets`: Logos and branding assets.

## 🚀 Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## 🎨 UI Features

- **Dynamic Workspace**: Grid-based project management with real-time renaming and sorting.
- **Interactive Chat**: Specialized BA Agent interface for requirement gathering.
- **Responsive Output**: Markdown-rendered technical specifications with one-click clipboard export.
